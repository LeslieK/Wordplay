WORDPLAY mini-project

To run this script, type:
$ run Wordplay.py

dependencies: WordplayUtils.py

From "sowpods.txt" which ontains a list of words:

find the word(s) with the longest repeated string
find words that contain UU
find words that contain Q not followed by U
find letters that are not doubled
find word(s) with the longest palindrome
find the most frequent letters and the words they are in
find the longest word/anagrams
find words that contain all vowels, in order
find words that contain all vowels, in any order (large printout!)
find the max frequency of each letter, and their words (large printout!)
